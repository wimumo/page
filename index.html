Ejemplos rápidos

Sketches de Processing, Patches de PureData y programas de Construct3 para comenzar a usar WIMUMO:

Los sketchs se pueden usar instalando Processing y para correr cualquiera de ellos es necesario instalar la librería oscP5. Los archivos o "patches" de Puredata fueron probados en la variante llamada Purr Data. Los archivos de Construct 3 se cargan en su editor online y debe correrse la aplicación de escritorio de WIMUMO en la misma PC.

  • [ejemplo-basico-cuadrados.pd] : Un Sketch que recibe mensajes de dos canales y controla el tamaño de dos cuadrados a partir del esfuerzo
  • [muestra-valores.pd] : Un Patch que recibe mensajes e imprime los valores.
  • [barras.c3p] : Un programa que modifica la altura de dos barras en función del esfuerzo.
  • [fly.c3p] : Un juego para controlar un personaje con el esfuerzo muscular.

Proyecto WIMUMO

WIMUMO es un dispositivo capaz de adquirir señales biopotenciales del cuerpo y transmitirlas a través de una conexión WiFi. (su nombre proviene de "plataforma de adquisición WIreless MUltiMOdal")

Específicamente, mide señales electromiográficas y electrocardiográficas (de los músculos y el corazón respectivamente). Está diseñado como una plataforma de bajo costo, fácil de construir y fácil de usar.

Nació como un dispositivo para performances artísticas, midiendo las señales producidas por el cuerpo del intérprete y transmitiéndolas mediante el protocolo OSC para producir imágenes y sonido.

Más información sobre este proyecto en gibic.ar/wimumo.

Repositorios

El proyecto consta de:

  • Una plataforma de hardware construida con componentes electrónicos de bajo costo como el ESP32, el ahora casi antiguo amplificador LM324 y suministros textiles comunes. [repositorio aún no publicado].
  • Gabinetes para impresión 3D y accesorios. Repositorio de gabinetes.
  • Firmware programado en el IDE de Arduino utilizando bibliotecas ESP32 de alto y bajo nivel para proporcionar una conexión a Internet simplificada, medición de señal, procesamiento y entrega a cualquier IP local y dirección de puerto utilizando el protocolo OSC. Repositorio del Firmware de Arduino.
  • Software programado en Electron para contar con una aplicación de escritorio opcional que permite observar las señales medidas, y utilizar algunas características básicas de WIMUMO. Repositorio de la app de escritorio.
